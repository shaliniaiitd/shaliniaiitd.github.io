# Deploy to shaliniaiitd.github.io

This folder is ready for GitHub Pages.

1. Create a GitHub repository named `shaliniaiitd.github.io`.
2. Copy the contents of this `portfolio` folder into the root of that repository.
3. Push to the repository's `main` branch.
4. GitHub Pages will serve the site at `https://shaliniaiitd.github.io/`.

Use `assets/profile-placeholder.png` as a temporary profile visual. Replace it with a real photo named `profile-photo.jpg` or update the image path in `index.html`.
